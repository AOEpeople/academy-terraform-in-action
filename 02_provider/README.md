# Provider

## azurerm Provider

```terraform
# providers.tf
terraform {
  required_providers {
    azurerm = {
      source = "hashicorp/azurerm"
      version = "3.77.0"
    }
  }
}

provider "azurerm" {
  features {}
}
```
_[-> providers.tf](providers-1.tf)_

```shell
terraform init
```
